# sistem-stok-barang
Manajemen Inventaris (Sistem Stok)
Aplikasi ini dirancang untuk memantau arus barang masuk dan keluar secara real-time. Sistem ini memberikan visibilitas penuh terhadap jumlah stok barang, meminimalisir kesalahan pencatatan, dan mempercepat proses pelaporan data inventaris.

Fitur Utama:
- Dashboard Interaktif: Memantau total barang, stok tersisa, dan ringkasan transaksi terbaru.
- Manajemen Transaksi: Pencatatan otomatis untuk setiap barang masuk dan keluar.
- Pelaporan: Fitur pelaporan data stok untuk kebutuhan audit atau evaluasi.
